# Event Management System

 Project under development 🚧

This project is being developed in Java with the goal of practicing Object-Oriented Programming (OOP), code organization, and good software development practices.

## About the Project

The system aims to manage events and tickets, allowing event registration, ticket control, and the application of business rules.

Throughout the development process, the following concepts will be applied:

- Object-Oriented Programming (OOP)
- Inheritance
- Polymorphism
- Encapsulation
- Interfaces
- Generics
- Collections
- Lambda Expressions
- Stream API
- Exception Handling

## Project Structure

The project is being organized following a separation of responsibilities:
application
└── Program

entities
├── Event
├── Ticket
├── CommonTicket
├── StudentTicket
├── VipTicket
└── enums

repositories
├── Repository
└── EventRepository

services
├── EventService
└── TicketService

exceptions
└── Custom Exceptions
